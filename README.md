# 🎬 Movie Recommender 5.0

Production-ready movie recommendation SaaS built with Python + Streamlit.

## Features
- Hybrid ML recommendations
- Similar movie search
- User ratings profiles
- Streaming availability
- Social buzz dashboard
- YouTube review links
- Metrics dashboard
- Render deployment ready

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app/app.py
```

## Deploy on Render
This repo includes `render.yaml`. Push to GitHub, connect on Render, deploy.
